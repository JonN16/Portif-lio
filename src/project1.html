Atletas do Mundo

Este foi um dos meus primeiros projetos utilizando HTML, CSS e JavaScript. O objetivo principal era criar uma funcionalidade que permitisse realizar buscas em um arquivo JSON, simulando o funcionamento de uma API. Desenvolvi minha própria API simples para exibir os dados de jogadores com base nas pesquisas realizadas. Embora seja um projeto básico, ele foi fundamental para aprimorar minhas habilidades em JavaScript e compreender melhor a integração entre dados e interface.
